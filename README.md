# First-Project - Titanic-Dataframe
## Project Overview:
Analysis on the Titanic dataframe with the objective to analyze factors affecting passenger survival rates, using exploratory data analysis. 
Dataset used: Seaborn Titanic Datset, with 891 passengers. 

## Key Insights:
  - Woman and children had significantly higher survival rates, which might suggest the following of the "women and children first"         rule.
  - First-class passengers were more likely to survive, suggesting a priority in evacuation.
  - Passengers older than 80 years had 0 chance of survival.

## Tools used:
  - Python
  - Pandas
  - Seaborn
